# Adidas Shoes Store App

## 1. Project Name
Adidas Shoes Store Mobile Application

---

# 2. Description

This is a mobile application for selling Adidas shoes.

Users can:
- Register and log in
- Browse Adidas shoes
- Search and filter products
- View product details
- Add products to cart
- Place orders
- Request returns/refunds
- Add products to wishlist
- Review purchased products
- Track shipping status
- Use AI assistant to choose suitable shoes and sizes

The application is designed for both Android and iOS platforms.
